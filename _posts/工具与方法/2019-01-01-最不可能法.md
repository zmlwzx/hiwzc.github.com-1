通过如何做到最不可能来激发创意，其步骤为：

1. 考虑与问题相关的产品、概念、特征特点等相关要素；
2. 根据具体情况进行极限假设；
3. 根据极限假设进行修改，使其具有可实现性。

举个例子：如何使餐馆盈利？

| 极限假设         | 解决方案 |
| ---------------- | -------- |
| 客人只点最贵的菜 |          |
| 一点完菜就做好   |          |
