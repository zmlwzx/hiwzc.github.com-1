## 计算 MD5

```shell
md5 file
```

## 加密与解密zip

```shell
zip -e file.zip file.txt
zip -e file.zip file1.txt file2.txt
zip -er director.zip director
zip -er director.zip director1 director2
unzip file.zip
```
